## Getting Started

First, install dependencies:

```bash

npm install
# or
yarn install
# or
pnpm install
# or
bun install

```
run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

for testing run

```bash
npm run test
# or
yarn test
# or
pnpm test
# or
bun test
```

Open [http://localhost:5173/](http://localhost:5173/) with your browser
